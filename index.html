Tipografía

La tipografía en CSS se controla principalmente con propiedades como font-size, font-family y font-weight.

Unidades: px, %, em, rem

  • px: píxeles. Medida fija. Ejemplo: font-size: 16px;
  • %: porcentaje respecto al tamaño del elemento padre. Ejemplo: font-size: 120%;
  • em: relativo al tamaño de fuente del elemento padre. Ejemplo: 1.5em es 1.5 veces el tamaño del padre.
  • rem: relativo al tamaño de fuente del elemento raíz (html). Ejemplo: 1rem = font-size del html.
/* Ejemplo de tamaños con px, em y rem */
html {
  font-size: 16px;          /* 1rem = 16px */
}

p {
  font-size: 16px;          /* 16px = 1rem */
}

p.grande {
  font-size: 1.5rem;        /* 1.5 * 16px = 24px */
}

p.pequeño {
  font-size: 0.8em;         /* 0.8 * tamaño de la fuente del padre */
}

Fuentes: font-family, font-weight

  • font-family: define la familia tipográfica (Arial, Verdana, etc.).
  • font-weight: define el grosor de la fuente (normal, bold, 300, 400, 700...).
/* Ejemplo de tipografía */
body {
  font-family: "Poppins", Arial, sans-serif;
}

h1 {
  font-family: "Georgia", "Times New Roman", serif;
  font-weight: 700;       /* negrita */
}

p.destacado {
  font-weight: bold;      /* equivalente a 700 en muchos casos */
}

Color

En CSS podemos escribir colores de varias formas. Las más usadas son Hex, rgb() y rgba().

Formatos: Hex, rgb(), rgba()

  • Hexadecimal (Hex): #RRGGBB, por ejemplo #FF0000 (rojo).
  • rgb(): combina rojo, verde y azul con valores de 0 a 255, por ejemplo rgb(255, 0, 0).
  • rgba(): igual que rgb() pero con un canal alfa (opacidad) de 0 a 1, por ejemplo rgba(255, 0, 0, 0.5).
/* Ejemplos de color */
h1 {
  color: #1D3557;               /* Hex */
}

p.importante {
  color: rgb(220, 20, 60);      /* rgb rojo intenso */
}

.caja-transparente {
  background-color: rgba(0, 0, 0, 0.3);  /* negro al 30% de opacidad */
}

Modelo de Caja

En CSS, cada elemento se representa como una caja. Esa caja tiene:

  • Content: el contenido (texto, imagen, etc.).
  • Padding: espacio interno entre el contenido y el borde.
  • Border: el borde de la caja.
  • Margin: espacio externo entre la caja y otras cajas.
/* Ejemplo de modelo de caja */
.caja-ejemplo {
  margin: 20px;            /* margen externo */
  border: 3px solid #1D3557; /* borde */
  padding: 15px;           /* espacio interno */
}
Esta es una caja de ejemplo con margin, border y padding.

Nota: por defecto, el ancho de un elemento cuenta el contenido + padding + border. Se puede cambiar con box-sizing: border-box;.

Display

La propiedad display define cómo se comporta una caja en el flujo de la página.

block, inline, inline-block

  • block: ocupa todo el ancho disponible y empieza en línea nueva (ej: <div>, <p>, <h1>...).
  • inline: ocupa solo el ancho de su contenido, no empieza en línea nueva (ej: <span>, <a>...).
  • inline-block: se comporta como inline (puede ir en línea con otros) pero se puede dar ancho, alto, margin, etc. como a un bloque.
/* Ejemplos de display */
div {
  display: block;          /* por defecto */
}

span {
  display: inline;         /* por defecto */
}

.menu-item {
  display: inline-block;   /* se pueden alinear en fila y darles ancho/alto */
}

Position

La propiedad position controla cómo se coloca una caja en la página.

  • static: posición por defecto, sigue el flujo normal de la página.
  • relative: se mueve respecto a su posición original.
  • absolute: se posiciona respecto al ancestro posicionado más cercano.
  • fixed: se fija respecto a la ventana del navegador (viewport).
/* Ejemplos básicos de position */
.caja-static {
  position: static;        /* valor por defecto */
}

.caja-relative {
  position: relative;
  top: 10px;
  left: 20px;
}

.caja-absolute {
  position: absolute;
  top: 0;
  right: 0;
}

.caja-fixed {
  position: fixed;
  bottom: 0;
  right: 0;
}
